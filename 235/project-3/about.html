Random dungeon project!

features:

  • randomly generated every time: procedurally generated so that no 2 games look the same
  • different enemy types: charging enemy, patroling enemy, and ranged enemy
  • TREASURE!
  • bonus areas: a big treasure will be on the opposite side of a room full of enemies that shoot stuff at you. navigate the room without getting shot to get the big cash money wad!

this game will be a top down tile map based adventure game where you go through a procedurally generated labyrinth that expands as you play.
this is done by creating an empty 2d array that starts with just the one map object, which acts as 0,0. as you explore the map,
the 2d array is added to in such a way that the 2d array will eventually become a map of it's own, for the dungeon itself.

this will be a 25x25 tile map for each room, with water traps that slow your speed and locked areas that require a key item somewhere else in the room. some items will incllude:

  • extra hearts
  • amulet of increased value: double the value of treasure when you have this item
  • immortality crystal! revive once at full health if you die in the game.
  • key! (unlocks doors)
  • skeleton key! (super rare)
  • swimming flippers (negates the slow effect in water)
note: unfortunately since this is a randomly generated map, it's kinda hard to make sketches of it.
